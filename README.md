### How to install
* Clone the repo from this website, then
* In command line:
  * ```$ git clone https://github.com/dumaevrinat/lantana.git```
  * ```$ cd lantana```
  * ```$ npm install```
  * ```$ npm start```

At this point you should see the app running on 

```http://localhost:3000```
